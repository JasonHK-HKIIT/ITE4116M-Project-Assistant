# AI Chat

The AI Chat feature lets you have a natural-language conversation with your personal AI assistant. You can ask questions, request information, and receive answers all within a single chat thread.

## Starting a New Chat

1. Click **New Chat** in the left sidebar.
2. A new, empty chat thread opens on the right.
3. Type your message in the text box at the bottom of the screen.
4. Press **Enter** or click the **Send** button to submit your message.

The assistant will respond in a few seconds.

## Continuing an Existing Chat

All your previous conversations are listed in the left sidebar under **Chats**.

1. Click on any chat in the list to open it.
2. Scroll up to review earlier messages.
3. Type a new message at the bottom to continue the conversation.

## Asking the Assistant

You can ask the assistant about anything it has access to. Here are some example questions:

- *"What classes do I have this week?"*
- *"Are there any badminton activities on campus?"*
- *"What is the latest news from VTC?"*
- *"Show me my profile."*
- *"What time is it now?"*

The assistant uses a set of tools (such as the Timetable tool and News tool) to fetch accurate, up-to-date information from MyPortal and present it in a readable format.

## Uploading Files

You can upload documents to give the assistant additional context. Supported file types include PDF, DOCX, TXT, and more.

1. Click the **paper clip** (attachment) icon in the typing area.
2. Select one or more files from your device.
3. The files are uploaded and indexed. You can then ask the assistant questions about the content of those files.

> **Note:** Uploaded files are associated with the current chat thread. They will be available as context for all subsequent messages in that thread.

## Chat History

Each chat thread is saved automatically. You can:

- Return to a previous chat at any time by clicking it in the sidebar.
- Rename a chat by clicking the pencil icon next to its name.
- Delete a chat by clicking the trash icon — this action cannot be undone.

## Keyboard Shortcuts

| Action | Shortcut |
| ------ | -------- |
| Send message | **Enter** |
| New line in message | **Shift + Enter** |

## Troubleshooting

**The assistant says it cannot find information.**
Try rephrasing your question. If you are asking about your timetable or profile, make sure your MyPortal account is properly linked.

**The assistant is taking a long time to respond.**
The assistant may be fetching data from MyPortal. Please wait a moment. If the issue persists, refresh the page and try again.

**My message was sent but there is no reply.**
Check your internet connection. If the problem continues, start a new chat thread.
