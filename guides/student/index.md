# ITE4116M Project Assistant — Student User Guide

Welcome to the **ITE4116M Project Assistant**, your AI-powered companion for study and campus life. This guide gives you an overview of everything the assistant can do for you as a student.

## What Can the Assistant Do?

The Project Assistant brings together several helpful features in one place:

| Feature | What it does |
| ------- | ------------ |
| [AI Chat](./chat.md) | Ask the assistant any question in plain language |
| [Timetable](./timetable.md) | Check your personal class schedule and upcoming events |
| [Student Activities](./student-activities.md) | Discover and search for campus activities |
| [News & Announcements](./news.md) | Stay up to date with VTC and institutional news |
| [My Profile](./profile.md) | View your personal profile and account details |

## Getting Started

1. Open the Project Assistant in your browser.
2. You will see the chat interface on the right and your conversation history on the left.
3. Select an existing chat or start a new one by clicking **New Chat**.
4. Type your question in the text box at the bottom and press **Enter** (or click the send button).

The assistant will reply with information drawn from the tools it has access to, such as your timetable or the latest news from MyPortal.

## Tips for Best Results

- Be specific in your questions. For example, *"What classes do I have on Monday?"* will give a better answer than *"What do I have?"*
- You can ask follow-up questions in the same chat — the assistant remembers the context of the conversation.
- If the assistant does not have the information you need, try rephrasing your question or providing more detail.
- Each chat thread is saved automatically so you can return to it later.
