# Timetable

The Timetable feature lets you ask the AI assistant about your personal class schedule and other calendar events sourced directly from MyPortal. You do not need to navigate to a separate page — simply ask the assistant in the chat.

## Asking About Your Timetable

In the chat, type a question such as:

- *"What classes do I have today?"*
- *"Show me my timetable for this week."*
- *"Do I have any classes on Friday?"*
- *"What is my schedule from 10 April to 14 April?"*

The assistant will retrieve your timetable events from MyPortal and display them in the chat.

## Understanding the Results

Each timetable entry shows:

| Field | Description |
| ----- | ----------- |
| **Event title** | The name of the class or event |
| **Type** | The type of event (e.g. lesson, examination, holiday) |
| **Source** | The system that generated the event (e.g. calendar event) |
| **Time** | The start and end date and time |
| **Location** | The room or venue |
| **Instructor** | The instructor responsible for the class |

### Example Response

> **ITE4116M Project**
> - Type: lesson
> - Source: calendar_event
> - Time: 2024-10-07 09:00 to 2024-10-07 10:30
> - Location: Room 101, Block A
> - Instructor: Dr Chan

## Filtering by Date Range

You can specify a date range to narrow down the results:

- *"Show my timetable from 7 October to 11 October."*
- *"What events do I have next Monday?"*

If you do not specify dates, the assistant will return events for the current day or a near-future range.

## Frequently Asked Questions

**Can I see my full semester timetable?**
Yes. Ask the assistant to show your timetable for a specific date range that covers the whole semester.

**Why are some events missing from my timetable?**
Timetable data comes directly from MyPortal. If an event is missing, it may not have been added to the system yet. Contact your programme administrator if you believe an event is incorrect.

**The times shown look wrong. What should I do?**
All times are displayed in Hong Kong Time (UTC+8). If the times still appear incorrect, please contact support.

**Can I export my timetable?**
Currently, the timetable is displayed in the chat only. You can copy the text to save it manually.
