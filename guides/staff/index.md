# ITE4116M Project Assistant — Staff & Admin User Guide

Welcome to the **ITE4116M Project Assistant**, an AI-powered conversational assistant backed by MyPortal data. This guide gives staff and administrators an overview of the available features and how to manage and configure the system.

## What Can the Assistant Do?

The Project Assistant provides both conversational and administrative features:

| Feature | What it does |
| ------- | ------------ |
| [AI Chat](./chat.md) | Converse with the AI assistant in plain language |
| [Configure Assistant](./configure-assistant.md) | Create and customise AI assistant profiles |
| [Manage Threads](./manage-threads.md) | Review and organise chat conversation threads |
| [News & Announcements](./news.md) | Access the latest VTC and institutional news |
| [My Profile](./profile.md) | View your account and staff profile details |

## Getting Started

1. Open the Project Assistant in your browser and log in with your staff MyPortal credentials.
2. The main interface shows the chat panel on the right and your conversation history on the left.
3. Use the **Configure** section (accessible from the top navigation) to create or manage assistant configurations.
4. Use the **Chats** section to start new conversations or review existing threads.

## Role and Permissions

Staff and administrator accounts have access to all the features listed above. Depending on your assigned role in MyPortal, you may also have access to:

- Creating **public** assistant configurations that are visible to other users.
- Managing conversation threads across the platform.

If you need elevated permissions, contact your system administrator.
